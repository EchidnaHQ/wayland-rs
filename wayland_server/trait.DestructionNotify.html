Expand description

A trait to be implemented on user data, which indicates when an object has been destroyed by a client.

Provided methods

[src]

fn object_destroyed(&self, _client_id: ClientId, _object_id: ObjectId)

Called when the object this user data is associated with has been destroyed.

Note this type only provides an immutable reference, you will need to use interior mutability to change the inside of the object.

Typically a Mutex would be used to have interior mutability.

You are given the ObjectId and ClientId associated with the destroyed object for cleanup convenience.
